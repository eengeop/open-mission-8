주식 가격 크롤링 웹
==============
***
> ## 프로젝트 개요
>- [x] Flask 기반 웹 서버에서 실시간으로 주식 정보를 크롤링 해온다.  
>- [x] 구입 금액에 알맞는 n장의 자동 로또 번호를 발행한다.
>- [x] 사용자로부터 당첨 번호 6자리와 보너스 번호 1개를 입력 받는다.
>- [x] 입력 받은 당첨 번호와 보너스 번호를 자동으로 발행된 번호와 비교해 일치하는 갯수만큼 당첨금을 지급한다.
>- [x] 최종 수익률을 계산한다.
***
## 기능 요구 사항
* 일정 주기로 주식 정보를 갱신 해야 한다.
* 즐겨찾기 기능이 존재한다.
* 즐겨찾기 종목은 SQLite DB에 저장된다.
* AJAX를 이용해 화면 부분 새로고침이 가능 해야 한다.
***
## 주요 기능 목록
***
### 새로 고침 시간 설정
- [x] 기본 새로 고침 시간(5초) 이외에 사용자가 지정하여 새로 고침 할 수 있다.
    + **예외 상항**
  - [x] 자연수가 입력되지 않을 시 예외 처리
  
### 주식 정보 크롤링
- [x] 설정된 시간 간격으로 주식 정보를 크롤링 해온다.
    + **예외 상황**
  -[x] 주식 정보를 가져올 수 없을때 예외처리

### 즐겨찾기
- [x] 원하는 종목을 즐겨찾기에 추가할 수 있다.
- [x] 즐겨찾기에 추가 된 종목을 즐겨찾기에서 지울 수 있다.
- [x] 즐겨찾기 된 주식들을 모아볼 수 있다.ㄴ
